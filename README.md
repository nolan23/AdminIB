# AdminIB
Admin panel for IB (Ijin Bermalam) System

- Requirement
	- dotnet versi >=2.2
- git clone https://github.com/nolan23/AdminIB.git
- cd AdminIB
- dotnet restore
- dotnet run
- open https://localhost:5001
- open file in Views/IB/Index.cshtml, change api url in get and put request   
- refresh the browser
